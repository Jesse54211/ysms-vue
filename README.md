## 仪器设备管理系统-前端
- 仪器设备管理系统前端项目基于vue、element-ui构建开发，实现[ysms](https://gitee.com/Jesse4023/ysms)后台管理前端功能
- 前后端分离，通过token进行数据交互，可独立部署
- 演示地址：[localhost:8001](localhost:8001) (账号密码：admin/admin)

## 实现功能
- 设备添加(上传图片，设备使用说明文档至OSS)
- 设备预约，设备报废，设备维修，定期维护
- 角色，权限管理，不同用户显示不同菜单
- 实验直播发布，观看
- 待添加...
